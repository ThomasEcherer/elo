Welcome to the ELO AS API Documentation!

On this side you will find information on using the ELO Standard APIs for your ELO AS standard libraries. It should help developers to handle standard ELO tasks more effectively.

For basic configuration and end user usage read the ELO AS manual.

Any questions about the ELO AS standard libraries can be written to the ELO Support Forum.
