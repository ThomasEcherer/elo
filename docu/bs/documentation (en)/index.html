ELO Business Solutions


ELO Business Solutions provide standardized concepts for the development of customized solutions. This documentation summarizes the basic aspects of solutions and shows which technological concepts are used in the scope of ELO Business Solutions.

The goal of this is to set new standards in terms of user experience, flexibility, and interaction with business applications. ELO Business Solutions build on the core ELO components and integrate seamlessly into ELOprofessional and ELOenterprise. Further, ELO Business Solutions are based on the proven multi-client strategy and can be used in the ELO Java Client and the ELO Web Client.

ELO Business Solutions components 1.03.002 installation

To start the installation, first log on to the ELO Java Client as an administrator.

From this page, the individual packages can be clicked and installed.

Alternatively, you can download the package from the ELO SupportWeb, which contains all files listed on this page, and install them individually.

1Basis technologies

Packages belonging to this solution are also linked on the solution's installation pages. The following packages can be installed for custom development/updates.

Each package may only be installed once. If a package is installed twice, it may lead to errors.

Business Solutions framework

This provides the class framework, namespaces, and architectural concepts for services, actions, and function modules.

Install 'common 1.03.002'

Exchange rates and foreign currency services

A service to request exchange rates for foreign currencies from the European Central Bank.

Execute the attached SQL scripts in the corresponding ELO repository database so that they are located in the same database as the other ELO tables.

Install 'common_fx 1.03.002'  Download SQL scripts

Monitoring services

Services to implement monitoring functions.

Install 'common_monitoring 1.03.002'

XML Importer services

Expanded XML Importer, which uses XPath mapping data to import data from all kinds of XML documents.

Install 'connector_xml 1.03.002'
